# **Codesharer 프로그램 요구명세서**

## **1. 프로그램 목적**

'Codesharer'는 정보 수업이나 코딩 협업 환경에서 발생하는 코드 공유의 번거로움을 해결하기 위해 설계된 웹 기반 실시간 코드 공유 프로그램입니다. 사용자는 모니터를 직접 보여주거나 복잡한 절차를 거치는 대신, 빠르고 간편하게 코드를 공유하고 소통함으로써 학습 효율성과 편의성을 높일 수 있습니다.

## **2. 주요 기능 요약**

1.  **실시간 공유방 생성 및 참여**
    *   **송신자:** 고유한 방 이름과 비밀번호를 설정하여 프라이빗한 공유방을 생성합니다.
    *   **수신자:** 전달받은 방 이름과 비밀번호를 입력하여 공유방에 안전하게 참여합니다.

2.  **실시간 코드 및 메시지 공유**
    *   **대화창:** 텍스트 기반의 대화창을 통해 실시간으로 메시지나 코드 스니펫을 주고받을 수 있습니다.

3.  **시각적 소통 및 파일 전송**
    *   **공유 화이트보드:** 마우스를 사용하여 다이어그램을 그리거나 코드에 대한 시각적인 설명을 덧붙일 수 있습니다.
    *   **파일/이미지 전송:** 버튼을 통해 소스 코드 파일이나 스크린샷 이미지 등을 직접 전송할 수 있습니다.

## **3. 사용자 설명**

*   **송신자 (Sender):** 코드를 공유하고자 하는 사용자입니다. 공유방을 생성하고, 공유할 코드, 파일, 그림 등을 전송하는 주체입니다.
*   **수신자 (Receiver):** 코드를 받고자 하는 사용자입니다. 송신자가 생성한 방에 참여하여 공유된 콘텐츠를 확인하고 상호작용합니다.

## **4. 입/출력 데이터 구조 (간단)**

*   **입력 (Input)**
    *   `방 생성 정보`: { 방 이름: String, 비밀번호: String }
    *   `메시지`: { 사용자: String, 내용: String, 시간: Timestamp }
    *   `화이트보드 데이터`: { 좌표: [x, y], 색상: String, 이벤트 타입: String }
    *   `파일 데이터`: { 파일 이름: String, 파일 타입: String, 데이터: Binary }

*   **출력 (Output)**
    *   `실시간 공유 화면`: 대화 내용, 화이트보드 그림, 파일 전송 내역이 실시간으로 동기화되어 표시됩니다.
    *   `다운로드 가능한 파일`: 전송된 파일을 클릭하여 다운로드할 수 있습니다.

## **5. 화면 또는 동작 시나리오**

1.  **[송신자]** Codesharer 웹사이트에 접속하여 '방 만들기' 버튼을 클릭합니다.
2.  **[송신자]** '방 이름'과 '비밀번호'를 입력하고 '생성' 버튼을 누릅니다. (예: 방 이름 - "mission1", 비밀번호 - "1234")
3.  **[송신자]** 생성된 방의 정보를 친구(수신자)에게 알려줍니다.
4.  **[수신자]** Codesharer 웹사이트에 접속하여 '방 참여하기'를 선택합니다.
5.  **[수신자]** 친구에게 받은 '방 이름'과 '비밀번호'를 입력하고 '참여' 버튼을 누릅니다.
6.  **[공통]** 송신자와 수신자의 화면에 공유 화이트보드와 대화창이 나타납니다.
7.  **[송신자]** 자신의 소스 코드를 복사하여 대화창에 붙여넣고 '전송' 버튼을 누릅니다.
8.  **[수신자]** 대화창에 송신자가 보낸 코드가 즉시 나타나는 것을 확인하고, 자신의 코드에 적용합니다.
9.  **[수신자]** 코드에서 이해되지 않는 부분을 화이트보드에 그리며 질문합니다.
10. **[송신자]** 화이트보드를 보며 추가 설명을 하거나, '파일 전송' 버튼을 눌러 관련 이미지 파일을 전송해줍니다.
